# Travel Packing List

**Pack Smart, Travel Light ✈️**

A beautiful, international-friendly packing list app to help travelers organize their luggage efficiently.

## Features

- 📦 **Smart Packing Lists** - Customized recommendations based on destination, season, and trip type
- 👥 **Multi-Person Collaboration** - Assign items to different travelers
- ⚖️ **Weight Management** - Track luggage weight with visual warnings
- 🛒 **Shopping List** - Mark items you need to buy
- 🌍 **International Support** - Covers global destinations with weather integration
- 📱 **Mobile Optimized** - Smooth card-based interface with swipe gestures
