请在Safari打开这个页面，然后：

  1. 打开开发者工具（Command + Option + I）
  2. 在Console里粘贴下面的代码：

步骤1：在test-scroll.html执行

const container = document.querySelector('.weather-scroll-container');
console.log('容器样式:', window.getComputedStyle(container));
console.log('容器宽度:', container.offsetWidth);
console.log('内容宽度:', container.scrollWidth);
console.log('是否可滚动:', container.scrollWidth > container.offsetWidth);

步骤2：在mobile.html执行

const container = document.querySelector('.weather-scroll') || document.querySelector('[style*="overflow-x: scroll"]');
console.log('容器样式:', window.getComputedStyle(container));
console.log('容器宽度:', container.offsetWidth);
console.log('内容宽度:', container.scrollWidth);
console.log('是否可滚动:', container.scrollWidth > container.offsetWidth);

对比这些数据：

  • 容器宽度（offsetWidth）
  • 内容宽度（scrollWidth）
  • scrollWidth > offsetWidth 必须是 true 才能滚动
